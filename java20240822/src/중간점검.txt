1. 클래스란?
 - 객체를 생성하기 위한 설계도

2. 메모리 구조:
 - 메소드 영역: 멤버변수, 클래스 변수, 상수, 생성자, 메소드 저장
 - 스택 영역:  지역변수, 매개 변수, 참조형 변수 저장
 - 힙 영역: 참조형 데이터 타입을 갖는 객체, 배열등  저장

3. 인스턴스 vs 클래스 변수, 인스턴스 vs 클래스 메소드 차이점
 - 인스턴스 변수: 클래스 내에 선언된 변수로 객체 생성시 새로운 변수가 생성, 
	클래스 변수와 달리 다른 객체와 공유되지 않음
 - 클래스 변수: 클래스 내부에 static으로 선언된 변수
	객체가 여러번 생성되어도 클래스 변수는 딱 1번만 생성되고 클래스 내부 전체에서 공유됨

 - 인스턴스 메소드: 인스턴스 생성 후 참조 변수.메소드명()으로 호출, (내부에서 인스턴스 변수, 클래스 변수 사용 가능 )
 - 클래스 메소드: 객체 생성없이 클래스명.메소드명()으로 호출, (내부에서 클래스 변수만 사용가능)

4. 인스턴스 변수 초기화 방법 2가지 기술?
 - 명시적 초기화
 - 생성자

5. 생성자 용도?
 - 객체의 초기화

6. this 용도 2가지 기술?
 - 클래스 생성자에서 내부 인스턴스 변수와 지역변수를 구분하기 위해 
 - 생성자가 다른 생성자를 호출 할 때

7. final 기능?
 - 변수 혹은 메소드에 붙여 사용할 수 있으며 해당 변수 혹은 메소드는 최초 1회 초기값 설정 후 값을 바꿀 수 없음
